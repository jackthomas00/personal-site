TradeCRE Encryption & Permissions

Security / Systems

End-to-end style encryption model for collaboration: per-user symmetric keys, encrypted document keys, and role-based sharing with minimal trust in the server.

  • Designed key hierarchy for users, groups, and documents.
  • Implemented client-side encryption flows integrated with API and database models.
  • Balanced security with usability for non-technical real-estate professionals.

Multi-Agent Traffic Light RL

RL / Simulation

Multi-agent reinforcement learning controller for traffic lights using SUMO, aiming to reduce congestion and stabilize flows under realistic traffic patterns.

  • Built training loop with custom reward functions and dynamic weighting.
  • Used Ray Tune to search over hyperparameters and policies.
  • Evaluated agents on realistic traffic patterns and bottlenecks.

Open Knowledge Compiler (OKC)

NLP / Knowledge

Prototype of a "knowledge substrate" that turns raw text into structured entities, claims, and relations, so agents can retrieve and reason over explicit facts instead of just embeddings.

  • Pipeline for entity extraction, claim detection, and relation typing.
  • Structured storage with pgvector for semantic search.
  • Focus on transparency and evidence-backed retrieval for agents.
